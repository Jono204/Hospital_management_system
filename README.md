# Hospital Management System

## Overview
A PHP and MySQL web application designed to manage hospital appointments and user authentication. The system allows administrators and users to book, update, view, and delete appointments.

## Features
- User login and authentication
- Book appointments
- Update appointments
- Delete appointments
- View appointments
- Admin user management

## Technologies Used
- PHP
- MySQL
- HTML

## Installation
1. Import `hospital_system.sql` into MySQL.
2. Place the project files in your web server directory.
3. Update the database credentials in `db_connect.php`.
4. Start Apache and MySQL.
5. Open `forms.html` in your browser.
